# Stopwatch Web Application ⏱️

## Task
Web Development Internship – Task 02

## Description
A stopwatch web application built using HTML, CSS, and JavaScript.
The application allows users to start, pause, reset the stopwatch,
and record lap times accurately. It provides a simple, interactive,
and user-friendly interface for measuring time intervals.

## Features
- Start, pause, and reset functionality
- Accurate time tracking (hours, minutes, seconds)
- Lap time recording
- Responsive and clean UI
- Interactive controls with smooth updates

## Tech Stack
- HTML
- CSS
- JavaScript
